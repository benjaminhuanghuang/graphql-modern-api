## Setup 
Creat project folder and npm init
```
    mkdir modern-api && cd modern-api && npm init -y
```
Install web framework 
```
npm i hapi -S
```
Install develop tools
``` 
    npm i nodemon -g
```
Mongoose for MongoDB accessing and ORM
```
    npm i mongoose -S
```
